## 1. 코드 수정

수정할 파일은 보통 **`swing_screener.py`** 하나입니다.

### 자주 바꾸는 것들

**① 실행 시간 변경** → `.github/workflows/daily_screen.yml` 의 `cron` 값
cron 은 **UTC 기준**(KST − 9시간)입니다.

| 원하는 시각 (KST)     | cron 값        |
| --------------------- | -------------- |
| 평일 17:30 (현재)     | `30 8 * * 1-5` |
| 평일 18:00            | `0 9 * * 1-5`  |
| 평일 16:00            | `0 7 * * 1-5`  |
| 매일(주말 포함) 17:30 | `30 8 * * *`   |

**② 종목 선정 기준 변경** → `swing_screener.py` 상단의 `FILTER` 부분

```python
FILTER = {
    "min_trade_amount":  10_000_000_000,   # 거래대금 최소 (100억)
    "min_market_cap":   100_000_000_000,   # 시가총액 최소 (1000억)
    "max_ext_ma20":     12.0,              # 20일선 대비 과열 차단 (%) ↑올리면 더 많은 종목
    "pullback_zone":    (-3.0, 7.0),       # 눌림목 이상 구간
    "rsi_zone":         (45.0, 68.0),      # 건강한 RSI 구간
    "top_n":            30,                 # 상위 몇 개 뽑을지
    ...
}
```

## 4. 바로 실행해서 확인하고 싶을 때

스케줄을 기다리지 않고 즉시 돌려보려면:

저장소 → **Actions 탭** → "📈 스윙매매 종목 스크리닝" → **Run workflow** 버튼

- `dry_run: true` → 구글시트 기록 없이 콘솔만 (테스트용)
- `dry_run: false` → 실제 시트에 기록
