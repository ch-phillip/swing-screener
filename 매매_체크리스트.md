# 📈 스윙매매 실전 체크리스트

> 풀백(눌림목) 스윙 전략 · 스크리너 기반 운용 가이드
> 핵심 원칙: **시장이 좋을 때 + 강한 종목을 + 눌림에 사서 + 손절은 칼같이**

---

## 🕐 1. 데일리 루틴 (언제 무엇을)

| 시점 | 할 일 |
|------|-------|
| **장 마감 후 (17시~)** | 스크리너 실행 (또는 GitHub Actions 자동 기록 확인) |
| **저녁 / 다음날 아침** | 구글시트에서 **내일 매매할 후보** 선별 |
| **다음날 장중** | 매수가 지정가 대응 → 진입 → 손절·익절 세팅 |
| **보유 중 매일** | 손절가 이탈 / 익절가 도달 / 보유일 초과 점검 |

> ⚠️ 이 스크리너는 **그날 종가**로 계산 → 나온 리스트는 **"내일" 매매용**이다.

---

## 📊 2. 시트에서 확인할 것 (우선순위 순)

1. [ ] **판단(K열)** = `✅ 진입후보` 인가? → 이것부터 거른다
2. [ ] **시장국면(S열)** = `🟢 위험선호` 인가? → 🔴면 매수 보류
3. [ ] **순위(Q열) / 종합점수(R열)** → 상위 + 7점↑ 우선
4. [ ] **매수가 · 손절가 · 익절가(M·N·O열)** → 주문 가격 메모
5. [ ] **섹터(P열)** → 오늘 「주도섹터」 탭과 겹치면 가점

---

## ✅ 3. 진입 전 체크리스트 (모두 Yes일 때만)

- [ ] 판단이 **✅ 진입후보** (= 🟢위험선호 + 종합 7점↑)
- [ ] 종목 시장이 **🟢 위험선호**
- [ ] **RS강세** (시장보다 강한 종목 / RS% 양수)
- [ ] 다음날 시초가가 **매수가 대비 갭상승 과하지 않음** (추격 금지)
- [ ] **손절가까지의 손실폭**이 감당 가능한 수준 (보통 −7~10%)
- [ ] 한 종목 비중이 **전체 자금의 정해둔 한도 이내** (분산)

> 하나라도 No → **그 종목은 패스**. 후보는 내일 또 나온다.

---

## 🎯 4. 진입 실행

1. **매수가(전일 종가)에 지정가 주문** 을 건다
   - 닿으면 체결, 안 닿으면 안 산다 → **갭 추격 자동 회피**
2. 체결 즉시 **손절가 주문(스톱)** 을 함께 세팅
3. **익절가** 를 목표로 기록 (손익비 1:2 구조)

> 시초가가 매수가보다 한참 위면 → 사지 말고 **눌림 기다리거나 스킵**.

---

## 🛡️ 5. 보유 중 관리 (기계적으로)

- [ ] **손절가 이탈** → 미련 없이 청산 (가장 중요)
- [ ] **익절가 도달** → 목표 달성, 일부/전량 청산
- [ ] **최대 보유 ~10일 초과** & 무반응 → 청산하고 자금 회전
- [ ] 보유 중 **시장이 🔴로 전환** → 신규 매수 중단, 기존 포지션 보수적으로

---

## 🚫 6. 절대 하지 말 것

- ❌ **손절가 무시 / 물타기** — 전략이 무너지는 1순위 원인
- ❌ **🔴위험회피 시장에서 무리한 매수** (백테스트상 불리)
- ❌ **갭상승 추격 매수** (손익비 붕괴)
- ❌ **종합점수만 보고 시장국면 무시**
- ❌ 진입후보 0개인 날 **억지로 매매** → 쉬는 것도 전략

## 📖 부록 A. 판단 등급의 의미

| 판단 | 조건 | 행동 |
|------|------|------|
| ✅ 진입후보 | 🟢위험선호 + 종합 7점↑ | 매수 검토 |
| ⚠️ 관심 | 🟢위험선호 + 5~6점 | 관찰, 7점 승격 대기 |
| ❌ 관망 | 🟢위험선호 + 4점↓ | 보류 |
| 🔴 관망(역장) | 🔴위험회피 시장 | 점수 무관 보류 |

## 부록 B. 종합점수(0~9) 구성

기본 5점(유동성·눌림목·RSI건강·거래량수축·추세강도)
\+ 신고가근접 + RS강세 + 박스돌파 + 거래량돌파

## 부록 C. 가격 산정 공식

- **매수가** = 전일 종가 (진입 기준가)
- **손절가** = 지지선(현재가·MA20 중 낮은 값) − ATR × 1.0
- **익절가** = 매수가 + (매수가 − 손절가) × 2  (손익비 1:2)

## 부록 D. 백테스트로 확인된 원칙 (참고용·생존편향 있음)

- 승률 약 56% · 손익비 약 1.7:1 → **구조적으로 기대값 플러스**
- **시장국면 필터가 잡매매를 줄여 성과 개선** → 그래서 역장 자동 관망 적용
- **무지성 7점 추격은 오히려 불리**, 단 "🟢시장 + 7점"이면 승률 최고(65%)
- **승자를 길게 끌수록 유리** (보유 5일<10일<20일)
